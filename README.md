# Real-Time Facial Emotion Detection System Using Deep Learning

## Overview

This project is a Deep Learning-based Emotion Recognition System that detects human emotions in real time using facial expressions and voice input. The system combines Computer Vision, Deep Learning, and Speech Analysis to identify emotions such as:

- Happy
- Sad
- Angry
- Fear
- Surprise
- Neutral
- Disgust

The project can be used in healthcare monitoring, online education, human-computer interaction, and mental health assessment.

---

## Features

- Real-time facial emotion detection using webcam
- Voice emotion recognition
- Deep Learning-based emotion classification
- Emotion logging and storage
- Interactive web interface using Flask
- Support for image and audio processing

## Installation

Clone the repository:

```bash
git clone https://github.com/kishothg18-a18y/Real-time-facial-emotion-detection-system-using-deep-learning.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```
